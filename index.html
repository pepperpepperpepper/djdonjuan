CAN ROBOTS MAKE ART?

DJ DON JUAN,
the art-making robot

DJ DON JUAN was a bot that created compositions using an image search and a server-side image editor.
He selected and edited the images independently using preset parameters.
In five days, he published over 4,000 images and 1,200 compositions.
He was built for use on the DUMP.FM website.
